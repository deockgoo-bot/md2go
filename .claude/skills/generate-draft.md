# /generate-draft

공문서 AI 초안을 생성합니다.

## 사용법
```
/generate-draft [문서종류] [목적/키워드]
```

## 예시
- `/generate-draft 기안문 "AI 도입 예산 신청"` — 기안문 초안 생성
- `/generate-draft 보고서 "스마트시티 1분기 추진 결과"` — 보고서 초안 생성
- `/generate-draft 공고문 "2026년 AI 바우처 사업 공모"` — 공고문 초안 생성

## 지원 문서 종류
기안문, 보고서, 공고문, 지시문, 협조문, 통보문, 조회문, 회신문, 계획서, 결과보고서

## 동작
1. 행정안전부 공문서 규정(두문·본문·결문) 적용
2. Claude API로 초안 텍스트 생성 (60초 이내)
3. JSON IR 경유 → HWPX 변환
4. 다운로드 링크 제공

## 연관 에이전트
doc-drafter 에이전트를 사용합니다.
